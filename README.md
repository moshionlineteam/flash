# flash

Some Flash games make use of `.swz` files, these are digitally signed libraries from Adobe. Some of their certificates have already expired and the games will fail to load. More of these will fail to run as we get closer to the end of the year.

These browser extensions have been patched to remove the certificate validity checks to keep the games playable.

Will include projectors soon too. :)

Flash version `34_0_0_330`
